# U.S. Sales Dashboard

## Project Overview
This Power BI dashboard analyzes U.S. sales data to provide insights into sales performance, profit, customer segments, and regional trends.

## Tools Used
- Power BI Desktop
- Microsoft Excel
- Power Query
- DAX
- Data Modeling

## Key Features
- Sales Analysis
- Profit Analysis
- Regional Performance
- Customer Segment Analysis
- Interactive Filters and Slicers
